🔧 SI L'APPLICATION NE DÉMARRE PAS
═══════════════════════════════════════════════════════════════════════════════

PROBLÈME 1 : Rien ne se passe quand je double-clique
────────────────────────────────────────────────────────────────────────────
  Cause probable : DLL Visual C++ Runtime manquantes
  
  Solution :
  1. Télécharger : https://aka.ms/vs/17/release/vc_redist.x64.exe
  2. Installer
  3. Redémarrer l'ordinateur
  4. Relancer l'application


PROBLÈME 2 : Windows Defender bloque l'application
────────────────────────────────────────────────────────────────────────────
  Solution :
  1. Ouvrir "Sécurité Windows"
  2. "Protection contre les virus et menaces"
  3. "Historique de protection"
  4. Si bloqué, cliquer sur "Autoriser"
  
  OU ajouter une exception :
  1. "Paramètres de protection contre les virus"
  2. "Gérer les paramètres"
  3. "Exclusions" → "Ajouter une exclusion"
  4. Sélectionner le dossier de l'application


PROBLÈME 3 : Antivirus tiers bloque l'application
────────────────────────────────────────────────────────────────────────────
  Solution :
  • Ajouter une exception dans votre antivirus
  • Ou désactiver temporairement l'antivirus pour tester


PROBLÈME 4 : Erreur "DLL manquante"
────────────────────────────────────────────────────────────────────────────
  Solution :
  • Installer Visual C++ Redistributable (voir PROBLÈME 1)
  • Vérifier que Windows est à jour


💡 UTILISATION DE L'APPLICATION
═══════════════════════════════════════════════════════════════════════════════

1. OBTENIR UN TOKEN WEBEX
   ────────────────────────────────────────────────────────────────────────
   • Aller sur : https://developer.webex.com
   • Se connecter avec votre compte Webex
   • Copier le token (commence par "Bearer ...")

2. DANS L'APPLICATION
   ────────────────────────────────────────────────────────────────────────
   • Coller le token dans le champ "Token Webex"
   • Cliquer sur "🔄 Charger les espaces"
   • Sélectionner les espaces à archiver
   • Cliquer sur "📦 Archiver la sélection"

3. RÉSULTAT
   ────────────────────────────────────────────────────────────────────────
   Les archives seront créées dans :
   C:\Users\[VotreNom]\Webex Archives\
   
   Format : Fichiers HTML avec tous les messages
